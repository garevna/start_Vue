Иконки фоновых изображений

Элемент aside нужно заполнить уменьшенными изображениями из массива fones

Создадим класс foneIco в файле css/styles.css:

  .foneIco {
      display: block;
      margin-top: 5px;
      width: 80px;
  }

Теперь можно использовать этот класс при вставке изображений в разметку ( index.html )

Для этого мы будем использовать уже знакомые нам директивы Vue.js v-for и v-bind:

  <body>
      <div id = "vueApp">
          <nav class = "navi">
              <img src = "css/photo_library.svg" class = "button"
                                @click = "showAside = !showAside"
              >
              <transition name = "aside">
                  <aside class = "aside" v-if = "showAside">
                      <img  v-for = "( item, index ) in fones"
                            :src = "item"
                            :key = "index"
                            class = "foneIco"
                      >
                  </aside>
              </transition>
          </nav>
          <div class = "figure"
               :style = '{ backgroundImage: currentBack }'>
               <img  v-for = "item in personages"
                     :key = "item.url"
                     :src = "item.url"
                     class = "personage"
                     :style = "{ top: item.y, left: item.x }"
                     v-on:click = "changeOrder"
               >
          </div>
      </div>
  </body>

Отлично! Наша панель выезжает при клике на активаторе, и в ней отображаются все фоновые изображения из массива fones
