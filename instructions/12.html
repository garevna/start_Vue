Выбор фонового изображения. Кнопка-активатор

Как помните, в реактивном объекте data экземпляра Vue есть свойство fones, которое является массивом ссылок на фоновые изображения, и свойство currentBack, в котором хранится ссылка на текущее фоновое изображение, отображаемое в представлении

Задача - обеспечить возможность выбирать фоновое изображение из массива fones, помещать выбранное значение в реактивную переменную currentBack, а остальное сделает реактивная система Vue.js

Вернемся к разметке ( файл index.html ) и вставим в контейнер приложения новый элемент - nav с классом navi

  <body>
      <div id = "vueApp">
          <nav class = "navi">
          </nav>
          <div class = "figure"
               :style = '{ backgroundImage: currentBack }'>
               <img  v-for = "item in personages"
                     :key = "item.url"
                     :src = "item.url"
                     class = "personage"
                     :style = "{ top: item.y, left: item.x }"
                     v-on:click = "changeOrder"
               >
          </div>
      </div>
  </body>

Цель - создать кнопку-активатор, которая будет находиться в левом верхнем углу, при клике на которой будет появляться панель с иконками фоновых картинок из массива fones

При клике на какой-либо иконке значение переменной currentBack должно меняться на значение соответствующего элемента массива fones

В файле css/styles.css ) создадим класс navi

.navi {
    position: fixed;
    top: 10px;
    left: 10px;
    bottom: 0;
    z-index:200;
}

Этот класс обеспечит положение кнопки-активатора в левом верхнем углу приложения

Теперь нужно создать саму кнопку-активатор

В принципе, не так важно, будет ли это элемент button, или кликабельное изображение / символ

Чтобы не подключать полностью библотеку Material Icons, воспользуемся уже имеющемся в нашем распоряжении файлом css/photo_library.svg

Вы можете использовать альтернативные изображения иконок css/menu.svg, css/style.svg или css/dashboard.svg

Итак, вставим изображение иконки в нашу разметку ( файл index.html ):

  <body>
      <div id = "vueApp">
          <nav class = "navi">
            <img src = "css/photo_library.svg" class = "button">
          </nav>
          <div class = "figure"
               :style = '{ backgroundImage: currentBack }'>
               <img  v-for = "item in personages"
                     :key = "item.url"
                     :src = "item.url"
                     class = "personage"
                     :style = "{ top: item.y, left: item.x }"
                     v-on:click = "changeOrder"
               >
          </div>
      </div>
  </body>

Очевидно, нужно создать класс button для иконки активатора, для чего переключимся на файл css/styles.css и добавим к его содержимому следующее:

  .button {
      cursor: pointer;
      width: 36px;
      background-color: #558855;
      padding: 4px;
      border-radius: 4px;
      box-shadow: 3px 3px 5px #000000aa;
  }

Нужно сделать иконку активатора кликабельной, чем мы и займемся далее
