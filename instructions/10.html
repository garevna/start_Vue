Динамическая привязка стилей

Мы уже использовали директиву v-bind для привязки стилей. Аналогично применим v-bind к атрибуту style каждого элемента массива personages:

  <body>
      <div id = "vueApp">
          <div class = "figure"
               :style = '{ backgroundImage: currentBack }'>
               <img  v-for = "item in personages"
                     :key = "item.url"
                     :src = "item.url"
                     class = "personage"
                     :style = "{ top: item.y, left: item.x }"
               >
          </div>
      </div>
  </body>

С помощью директивы v-bind мы динамически связываем атрибут style элемента с объектом { top: item.y, left: item.x }

Обратите внимание, что имена свойств этого объекта соответствуют именам css-атрибутов top и left

Теперь при изменении значений свойств x и y элементов массива personages у соответствующих элементов DOM в представлении будут изменяться значенния css-атрибутов top и left, и элементы будут менять свое положение в окне браузера

Ура, мы можем управлять из своей модели положением персонажей на странице

Однако нужно, чтобы координаты персонажей в окне браузера менялись при наступлении какого-то события
