Свойство data экземпляра Vue

У экземпляра Vue есть унаследованное свойство data, которое является объектом

  new Vue ({
    data: {

    }
  }).$mount( "#vueApp" )

В этом объекте будут храниться реактивные данные модели, т.е. данные, изменение которых будет отслеживать реактивная система Vue с целью обновления DOM по мере необходимости

Добавим в объект data два пустых массива:

  new Vue ({
    data: {
      fones: [],
      personages: []
    }
  }).$mount( "#vueApp" )

В массиве fones мы будем хранить ссылки на фоновые картинки, а в массиве personages - изображения персонажей

Данные для массивов fones и personages у нас уже есть в папке json файлах json/fones.json и json/personages.json

Нам нужно получить данные из файлов на сервере в массивы fones и personages
