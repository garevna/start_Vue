Создание копии репозитория на github

Если нет аккаунта на github.com, создате его

Заходим в рабочий репозиторий   и нажимаем  Fork

Теперь в вашем аккаунте есть копия этого репозитория

Установка git на локальном компьютере

Установщик спросит, добавлять ли в меню проводника возможность запуска файлов с помощью Git Bash ( консоль Git ) и GUI (графический интерфейс). Подтвердите действие, чтобы далее работать через консоль в Git Bash, остальное можно оставить по умолчанию

Создание локальной копии репозитория

Запускаем консоль git Bush (MinGW 64)

В консоли переходим в папку, куда хотим склонировать репозиторий

( в примере это папка test-git-clone )

cd test-git-clone

Клонируем репозиторий с помощью команды git clone [url репозитория]

Теперь у вас есть локальная копия репозитория, с которой мы и продолжим работу
