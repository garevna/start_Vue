В нашем SPA библитека Vue.js уже подключена в head файла index.html

<script src = "https://unpkg.com/vue"></script>

Создание экземпляра Vue

Перейдем в файл script.js

Экземпляр объекта Vue создается с помощью конструктора, которому передается объект параметров

Для начала создадим экземпляр Vue без параметров

  new Vue ({

  })

Мы создали объект экземпляра с унаследованными от прототипа свойствами и методами

Унаследованные свойства и методы экземпляра имеют идентификаторы, начинающиеся с символов $ или _

Созданный с помощью конструктора Vue объект - это модель, которая пока еще не реактивна, потому что никак не связана с DOM

Вернемся к файлу index.html и создадим элемент DOM, который будет точкой монтажа нашей модели:

  <body>
      <div id = "vueApp"></div>
  </body>

Вернемся к файлу script.js

Теперь можно связать нашу модель с представлением, т.е. элементом DOM с id = "vueApp"

Для этого используем унаследованный метод экземпляра $mount

  new Vue ({

  }).$mount( "#vueApp" )

Итак, мы создали модель, связали ее с представлением, теперь можно добавлять свойства и методы модели
