Конечная цель

Наша цель - создать одностраничное приложение (SPA), где будет динамически обновляемый фон и произвольно перемещающиеся по этому фону персонажи

Ссылки на фоновые изображения и изображения персонажей будут храниться в JSON-файлах на сервере, откуда мы должны эти данные получить

Для построения SPA мы воспользуемся UI-фреймворком Vue.js

Для загрузки данных с сервера подключим плагин vue-resource

В остальном - да поможет нам JavaScript ☺

Обновление фона должно происходить при выборе пользователем фоновой картинки из предлагаемого "ассортимента" фоновых изображений

Персонажи должны быть случайным образом "разбросаны" в пределах видимой части окна браузера, и при клике на любом персонаже менять свое положение

Изменение положения персонажей должно происходить плавно
